# Tmux

## Session management

### From CLI

* `tmux list-sessions`
* `tmux new-session -s <session_name>`
* `tmux attach <session_name>`

### From inside session

* `Ctrl`-`b` `d` - Detach from current session

## Panes

* `Ctrl`-`b` `"` - Split current pane horizontally
* `Ctrl`-`b` `%` - Split current pane vertically
* `Ctrl`-`b` `x` - Close current pane
